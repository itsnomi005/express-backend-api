# Express Backend API

A RESTful API built with Express.js and MongoDB.

## Features
- User authentication
- CRUD operations for todos
- MongoDB integration

## Setup
```bash
npm install
node app.js
## API Endpoints
| Method | Endpoint       | Description          |
|--------|----------------|----------------------|
| GET    | /api/posts     | Get all posts        |
| POST   | /api/users     | Create a new user    |
